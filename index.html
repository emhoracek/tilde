I work on Haskell web applications for work and for fun! Here's my typical stack:

  • Fn -- routing framework
  • Larceny, Heist, or Blaze -- HTML templating
  • Aeson -- JSON generation and parsing
  • Opaleye -- SQL generation
  • postgres-simple -- SQL querying
  • Digestive Functors -- form processing and validation
